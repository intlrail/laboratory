# 06 - interactivity

## Keyboard commands

| Key | Result                   |
|---  |---                       |
| `1` | Click to grow mode       |
| `2` | Mouse trailer (on click) |
| `3` | Asteroids mode           |
| `4` | Radial Bust-a-Move mode  |
